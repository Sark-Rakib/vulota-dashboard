# 🛡️ Vulota Dashboard (Comapiens Assignment)

A modern, responsive admin dashboard built with React + Vite.  
This project is created as a frontend skill assessment focusing on UI/UX, component structure, responsiveness, and clean code architecture.

---

## 🚀 Live Features

- 📊 Dashboard Overview with stats cards
- 📈 Analytics section with charts & insights
- 👥 User Management (CRUD UI)
- 📑 Reports & Activity Logs
- 🛡️ Security Insights dashboard (offensive security themed UI)
- 📱 Fully responsive design (mobile + tablet + desktop)

---

## 🧰 Tech Stack

- ⚛️ React.js
- ⚡ Vite
- 🎨 Tailwind CSS
- 🧩 Lucide React Icons
- 📊 Recharts (if used for charts)
- 🧠 Static / dummy JSON data

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Sark-Rakib/vulota-dashboard.git
cd vulota-dashboard
npm install
npm run dev
```
